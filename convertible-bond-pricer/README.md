# convertible-bond-pricer

Prices convertible bonds under two models and compares them systematically
across synthetic scenarios to show when — and why — credit risk matters for hybrid instrument valuation.

## What it does

A convertible bond (CB) is simultaneously a risky corporate bond and an equity call option.
The key question is: *how should the equity component be discounted when the issuer may default?*

| Model | Equity discount rate | Credit-sensitive? |
|-------|----------------------|-------------------|
| **Black-Scholes** benchmark | Risk-free rate `r` | ✗ (only bond floor) |
| **Tsiveriotis-Fernandes** (1998) | `r + h` (hazard rate) | ✓ |

The TF model splits the CB into two coupled PDEs solved on a trinomial tree:
- `u` — total CB value (discounted at `r`)  
- `v` — cash-only component (coupons + face, discounted at `r + h`)

This correctly penalises the cash flows that depend on the issuer surviving,
while the equity component (conversion value) is discounted at `r`.

**Key result:** TF ≤ BS always. The divergence is largest in the *balanced regime*
(moderate stock price, moderate spread) and collapses at the extremes:
- *Equity regime*: conversion is certain, credit risk irrelevant → TF ≈ BS
- *Distressed regime*: bond floor dominates, both models agree on a risky bond price

## Structure

```
convertible-bond-pricer/
├── README.md
├── pyproject.toml
├── src/
│   ├── models.py          ← pricing engine (TF + BS, trinomial tree)
│   ├── synthetic_data.py  ← bond/scenario generators (no real data needed)
│   └── plots.py           ← all analysis figures
├── scripts/
│   └── run_analysis.py    ← end-to-end runner
├── tests/
│   └── test_models.py     ← pytest suite (structural, monotonicity, convergence)

    ├── main.tex           ← write-up

```

## Quick start

```bash
pip install -e ".[dev]"

python scripts/run_analysis.py
```

## Representative bonds

| Case | Description | Key parameter |
|------|-------------|---------------|
| Vanilla | No optionality, 4% semi-annual | Baseline |
| Callable | Issuer calls at years 2/3/4 | Reduces holder value |
| Putable | Holder puts at par at years 2/3 | Increases holder value |
| Distressed | Spread 15%, stock deep OTM | Bond-like (δ ≈ 0) |
| Equity-sensitive | Stock deep ITM, spread 1% | Equity-like (δ ≈ cr) |
